# Indicators of Compromise (IOC) Investigation


## Overview

Indicators of Compromise are pieces of evidence that may suggest malicious activity inside a network.

During this traffic analysis project, captured packets were reviewed for possible suspicious indicators.


---

# IOC Checks Performed


## Suspicious IP Connections


Checked:

- Unknown external IP addresses
- Repeated communication attempts
- Unexpected destinations


Result:

No suspicious external communication identified.



---

# Suspicious DNS Queries


Checked For:

- Random-looking domains
- Unknown domain requests
- Possible malware communication


Result:

Normal DNS resolution activity observed.



---

# Unusual Network Ports


Checked For:

- Unknown services
- High-numbered ports
- Unexpected connections


Result:

No unauthorized services detected.



---

# Possible Attack Indicators Reviewed


Reviewed indicators related to:


- Command and Control communication

- Port scanning

- Data exfiltration

- Abnormal connection attempts



---

# Conclusion


No confirmed compromise indicators were discovered.

This analysis demonstrates the investigation workflow followed during basic SOC traffic review.
