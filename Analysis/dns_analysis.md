# DNS Traffic Analysis


## Objective

Analyze DNS communication inside captured network traffic.


## Filter Used


dns


## Observations

DNS requests were identified between the client machine and DNS server.

The traffic contained:

- Domain name queries
- DNS responses
- IP address resolution


## Security Importance

DNS analysis helps security analysts detect:

- Malicious domain communication
- Suspicious external connections
- DNS tunneling attempts


## Finding

No malicious DNS activity was discovered during this analysis.
