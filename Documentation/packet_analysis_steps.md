# Packet Analysis Methodology


## Step 1: Capture Traffic

Captured live packets using Wireshark.


Collected:

- TCP packets
- DNS queries
- HTTP traffic
- ICMP packets


---

## Step 2: Identify Conversations

Reviewed communication between:

- Source IP
- Destination IP


Analyzed:

- Who initiated communication
- Which protocols were used


---

## Step 3: Protocol Analysis


Investigated:

DNS:

Domain resolution


TCP:

Connection establishment


HTTP:

Web communication


ICMP:

Network testing


---

## Step 4: Threat Review


Looked for:

- Unusual connections
- Failed communication
- Suspicious domains
- Unknown protocols


---

## Step 5: Documentation


Recorded:

- Observations
- Screenshots
- Findings
- Recommendations
